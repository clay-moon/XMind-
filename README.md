# XMind-tally-book

## 安装
### 项目地址
```
git clone git@github.com:clay-moon/XMind-tally-book.git
```
### 进入文件夹
```
cd xmind
```
### 安装依赖
```
npm install
```
### 运行开发版本
```
npm run serve
```
### 运行生产版本
```
npm run build
```
